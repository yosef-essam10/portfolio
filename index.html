[PROJECT_04]

Fire Detection & Segmentation

Pixel-precise fire localization using YOLO segmentation for disaster prevention

Challenge

Early fire detection is critical in preventing disasters. Traditional detection systems fail to precisely localize fire regions in real-time, limiting effective response capabilities.

Solution

Built a real-time fire segmentation system using YOLO instance segmentation to accurately detect and localize fire regions at pixel-level precision, enabling precise risk area estimation and potential fire spread monitoring.

Pixel-level fire segmentation with confidence scores showing precise region localization for disaster prevention

Why Segmentation?

  • Object Detection → bounding box approximation
  • Segmentation → exact fire region mask
  • Enables precise risk area estimation
  • Supports pixel-level analysis and monitoring
  • Better integration with fire spread prediction systems

Tech Stack

Python YOLOv8-Seg Instance Segmentation OpenCV Real-time Inference

[PROJECT_05]

RAG-Powered Chatbot

Domain-specific conversational AI using Retrieval-Augmented Generation

Challenge

Traditional chatbots provide generic answers and lack access to domain-specific knowledge, often resulting in hallucinations and unreliable responses.

Solution

Developed a lightweight domain-specific chatbot using Retrieval-Augmented Generation (RAG) powered by LangChain, enabling context-aware answers based on a custom knowledge base with significantly reduced hallucination.

Architecture Pipeline

  • User Query → Embedding Generation
  • Vector Search (Top-K relevant chunks from database)
  • Context Injection into LLM prompt
  • Generated Answer with source attribution
  • Conversational memory for multi-turn dialogue
